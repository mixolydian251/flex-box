Justify content

justify-content refers to alignment along the main axis of the parent div. By default this is along the horizontal axis.

common values

  • flex-start (default)
  • flex-end
  • center
  • space-evenly
  • space-between

align-items refers to alignment along the minor axis of the parent div. By default this is along the vertical axis.

common values

  • flex-end
